# GiantoDental Lab - Admin Dashboard

The **GiantoDental Lab Admin Dashboard** is a web-based application designed for managing dental lab operations, customer orders, and staff activities efficiently.  
This dashboard allows admins to manage service requests, monitor orders, update profiles, and interact with real-time data through a responsive interface.

## Key Features

- **Admin Profile Management**: Upload and edit profile photos, update personal information, and manage account details.  
- **Customer Order Management**: View, approve, or reject service requests submitted by clients via the mobile app.  
- **Real-Time Updates**: All data, including profile changes and order statuses, are updated instantly using Firebase Firestore.  
- **Secure Authentication**: Admin login system with session management and logout functionality.  
- **Responsive Design**: Optimized for desktop and tablet usage.  
- **Error Handling & Validation**: Input validation and comprehensive error messages ensure smooth operations.  

## Tech Stack

- **HTML, CSS, JavaScript** for front-end development  
- **Firebase Firestore** for real-time database  
- **Firebase Storage** for profile image uploads  
- Optional: **Material Icons** for UI enhancements
